# State

If we were not able to send the file to DATEV, we will set the state "FAILED". The other values are synced from DATEV for the respective import jobs.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `FAILED`       | FAILED         |
| `UPLOADED`     | UPLOADED       |
| `IMPORTED`     | IMPORTED       |
| `CORRUPTED`    | CORRUPTED      |
| `DELETED`      | DELETED        |
| `AUTO_DELETED` | AUTO_DELETED   |
